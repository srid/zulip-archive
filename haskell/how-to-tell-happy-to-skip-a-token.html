Can I tell Happy to always ignore a certain special token? I would like for it to always ignore the Empty token and I'd rather not filter the tokens before sending them to Happy, if I can help it.

data Token
  = String String
  | Colon
  | ColonEqual
  | Comma
  | LParen
  | RParen
  | Empty
  ...

%token
  string        { String $$ }
  ':'           { Colon }
  ':='          { ColonEqual }
  ','           { Comma }
  '('           { LParen }
  ')'           { RParen }
  <ignore>      { Empty }
  ...
