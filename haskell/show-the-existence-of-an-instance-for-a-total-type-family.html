there is a dumb trick to doing this

Sandy Maguire
2020-05-01 21:23:22
data SBool a where
  STrue :: SBool 'True
  SFalse :: SBool 'False

data Dict c where
  Dict :: c => Dict c

sfoo :: SBool a -> Dict (C a)
sfoo STrue = Dict
sfoo SFalse = Dict
Sandy Maguire
2020-05-01 21:23:33

sfoo is the proof that C is closed for booleans
