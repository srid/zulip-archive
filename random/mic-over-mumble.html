I don't have any usable microphone on my PC right now, but I found pretty ridiculous way of solving this - I've joined Mumble call with both PC and phone, and using https://github.com/pzmarzly/mic_over_mumble I've connected sound from call to a virtual input source which I could then use as a microphone :joy:

pzmarzly/mic_over_mumble
Use Android/iOS device as your PC's remote microphone, using Mumble + Plumble. Or use microphone of one PC on other PC. Linux-only script - pzmarzly/mic_over_mumble
